# Silver-OPCUA-Toolkit
Open-source industrial OPC UA platform for realtime monitoring and visualization.

## Overview

Modern open-source OPC UA toolkit for industrial monitoring, realtime visualization, and future AI-assisted industrial workflows.

The project focuses on providing modern tools for:

- OPC UA connectivity
- Realtime tag monitoring
- Industrial data visualization
- Web-based industrial interfaces
- Future AI-assisted industrial workflows

Built with a modern full-stack architecture using Python and React.

---
## Why Silver OPC UA Toolkit?

Most industrial software tools are outdated, difficult to use, and built around legacy workflows.

Silver OPC UA Toolkit aims to provide a modern, developer-friendly, and cross-platform industrial experience with clean UI, realtime workflows, and future AI integration.

## Vision

Silver OPC UA Toolkit aims to become a modern industrial data workspace for engineers working with OPC UA, industrial monitoring, diagnostics, and future AI-assisted workflows.

The long-term goal is to provide lightweight, modern, cross-platform industrial tools that replace outdated legacy industrial software experiences.

## Quick Start

### Requirements

- Python 3.13+
- Node.js 22+
- npm

---

### Clone Repository

```bash
git clone https://github.com/Armin22-Programmer/Silver-OPCUA-Toolkit.git
cd Silver-OPCUA-Toolkit
```

### Backend Setup
```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload
```

### Frontend Setup
```bash
cd frontend
npm install
npm run dev
```
